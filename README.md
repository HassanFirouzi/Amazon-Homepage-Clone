# Amazon Homepage Clone

This project has been developed to recreate the basic structure of Amazon’s homepage from scratch using HTML and CSS. The project adopts responsive web design to provide a smooth user experience across all devices (mobile, tablet, and desktop).

## The project includes the following elements:

Header and navigation bar

Homepage images

Footer section

Responsive design features
